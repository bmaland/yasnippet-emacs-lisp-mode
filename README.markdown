# Snippets for Emacs Lisp Mode

## About

Collection of Emacs Lisp snippets for
[yasnippet](http://code.google.com/p/yasnippet/ "yasnippet - Google Code").
Some of them are based on snippets in TextMate's Lisp bundle.
